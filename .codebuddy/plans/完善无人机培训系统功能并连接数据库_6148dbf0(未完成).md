## 产品概述

完善无人机培训系统的前后端功能，确保所有模块与 CloudBase 数据库正确连接，实现完整的数据流。

## 核心需求

1. 初始化/检查数据库集合结构
2. 部署和配置云函数
3. 确保前端服务层与后端 API 连接正常
4. 生成测试数据
5. 重新构建并部署应用

## 功能模块

- 用户端：首页、课程、购物车、订单、考试、证书、营销
- 管理后台：课程、学员、教师、排课、出勤、订单、财务、考试、证书、营销、系统设置

## 技术栈

- **前端**: React 19 + TypeScript + Vite + TailwindCSS + MUI
- **后端**: CloudBase (云函数 + NoSQL 文档数据库)
- **状态管理**: Zustand
- **HTTP 客户端**: Axios

## 实现方案

### 数据库架构

使用 CloudBase NoSQL 文档数据库，集合包括：

- `users`: 用户信息
- `courses`: 课程信息
- `orders`: 订单信息
- `teachers`: 教师档案
- `schedules`: 排课信息
- `exams`/`questions`: 考试和题库
- `certificates`: 证书信息
- `coupons`: 优惠券
- `studyProgress`: 学习进度

### 后端架构

云函数分层设计：

- `admin`: 通用 CRUD 管理接口
- `auth-login`: 用户认证
- `init-uav`: 系统初始化
- `api/*`: 业务 API 云函数

### 数据流

前端 Zustand Store → API 调用 → CloudBase SDK → 云函数 → 数据库

## 实现要点

1. 首先初始化数据库集合和安全规则
2. 部署核心云函数（admin, auth, init）
3. 验证前端数据库连接配置
4. 初始化测试数据
5. 构建并部署前端
