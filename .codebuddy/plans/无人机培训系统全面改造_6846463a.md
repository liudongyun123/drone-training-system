## 产品概述

全面升级无人机培训系统，实现多方式登录、精细化权限管理和响应式UI改造。

## 核心功能

### 1. 登录模块

- **游客登录**：匿名访问，仅浏览前台内容
- **微信授权登录**：微信扫码（PC端）+ 微信授权（微信内）
- **手机验证码登录**：短信验证码登录
- **账号密码登录**：保留传统登录方式

### 2. 角色权限系统（RBAC）

| 角色 | 权限范围 |
| --- | --- |
| 管理员 | 后台所有权限（课程/题库/学员/财务/系统设置） |
| 教师 | 课程管理、考试题库、学员管理、排课出勤 |
| 学员 | 前台所有权限，仅可访问已购买课程和对应题库 |
| 访客 | 仅浏览前台公开内容，无购买/学习权限 |


### 3. 响应式布局

- **断点设计**：手机(<768px)、平板(768-1024px)、PC(>1024px)
- **移动端优化**：底部导航栏、手势操作、图片懒加载
- **PWA支持**：离线缓存、添加到主屏幕

### 4. 全面功能检查

- 检查所有数据库集合字段名一致性
- 检查所有服务层函数调用正确性
- 检查前后端数据交互完整性

## 技术栈

- **前端框架**：React + TypeScript + Vite
- **状态管理**：Zustand
- **UI组件库**：MUI (Material-UI) + Tailwind CSS
- **认证服务**：CloudBase Auth v2
- **数据库**：CloudBase NoSQL
- **部署**：CloudBase 静态托管

## 架构设计

### 认证流程

```
用户选择登录方式 → CloudBase Auth → 获取Token → 查询用户角色 → 跳转对应页面
```

### 权限控制

- **路由级**：AuthGuard 组件控制页面访问
- **组件级**：PermissionWrapper 控制功能按钮
- **数据级**：服务端查询时过滤数据范围

### 响应式实现

- CSS Media Query + MUI useMediaQuery
- 移动端优先设计
- 动态加载不同布局组件

## 设计风格

采用**Editorial/Magazine**风格，专业、现代、有质感。

### 设计理念

- 清晰的信息层级
- 专业的视觉呈现
- 优雅的过渡动画
- 深色/浅色主题切换

### 布局策略

- 非对称网格布局
- 卡片式信息展示
- 悬浮操作按钮
- 沉浸式内容区

### 响应式断点

- **Mobile**: < 768px，单列布局，底部导航
- **Tablet**: 768px - 1024px，双列布局，侧边导航
- **Desktop**: > 1024px，多列布局，顶部导航

### 移动端适配

- 底部固定导航栏（首页/课程/考试/我的）
- 全屏模态框登录
- 手势滑动返回
- 触摸友好的按钮尺寸（最小44px）
