## 产品概述

优化管理后台侧边栏导航，反映模块合并后的新结构，提升视觉设计和用户体验。

## 核心变更

1. **菜单合并**：

- 排课管理 + 出勤管理 → 合并为"排课出勤"
- 订单管理 + 财务统计 → 合并为"订单财务"
- 考试管理 + 题库管理 → 已合并为"考试题库"（使用 AdminExamsUnited）

2. **视觉优化**：

- 侧边栏采用现代深色主题
- 菜单项添加图标和悬停效果
- 优化选中状态样式

## 目标文件

- `src/components/Layout.tsx` - 更新菜单配置和侧边栏样式
- `src/router/index.tsx` - 更新路由配置（添加重定向）

## 技术栈

- **框架**: React 19 + TypeScript
- **样式**: TailwindCSS + DaisyUI
- **图标**: Lucide React
- **路由**: React Router v6

### 2. 路由调整

- `/admin/schedules` - 排课出勤统一入口
- `/admin/attendance` → 重定向到 `/admin/schedules`
- `/admin/orders` - 订单财务统一入口
- `/admin/finance` → 重定向到 `/admin/orders`
- `/admin/question-banks` → 重定向到 `/admin/exams`

### 3. 侧边栏视觉优化

- 使用渐变背景替代纯色
- 选中项添加左侧指示条
- 优化图标和文字间距
- 添加平滑过渡动画

## 设计风格

采用**专业商务风格**，深色侧边栏配合精致的交互细节，体现管理后台的专业性和现代感。

### 设计要点

1. **侧边栏背景**: 深 slate-900 到 slate-800 的微妙渐变，增加层次感
2. **选中状态**: 蓝色渐变背景 + 左侧 3px 高亮指示条
3. **悬停效果**: 背景色渐变 + 图标轻微放大
4. **菜单分组**: 使用 subtle 的分隔线区分功能区域
5. **Logo区域**: 优化为更现代的卡片式样式

### 布局策略

- 保持左侧固定宽度 (w-64)
- 菜单项采用左对齐，图标+文字紧凑排列
- 底部"返回前台"使用独立区域，视觉分离
