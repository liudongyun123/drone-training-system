# 2026年3月20日 工作日志

## 🔴 问题诊断：429 Too Many Requests

### 根本原因
**`useAuthStore.ts` 的 `initAuth()` 函数在每次页面加载时都会调用 `authService.checkSession()`，导致大量重复的 CloudBase API 请求，触发 429 错误。**

### 错误代码位置
- 文件：`src/store/useAuthStore.ts`
- 问题代码：第 140 行 `await authService.checkSession()`
- 影响范围：所有页面加载时都会触发

### 修复方案
**重新调整初始化顺序，优先检查管理员会话（无需 API 调用），仅在必要时检查普通用户会话：**

1. 优先检查 `localStorage` 中的管理员会话（本地存储，无 API 调用）
2. 仅在无管理员会话且存在 `auth-storage` 时才调用 `checkSession()`
3. 移除不必要的 API 调用，减少请求频率
