### 8.1 必需的环境变量

请在 [云函数配置页面](https://console.cloud.tencent.com/scf/list-detail?rid=4&ns=rcwljy-5ghmq2ex26764978&funcname=mobile-auth) 配置:

| 变量名 | 说明 |
|-------|------|
| TENCENT_SMS_SECRET_ID | 腾讯云 SecretId |
| TENCENT_SMS_SECRET_KEY | 腾讯云 SecretKey |
| TENCENT_SMS_SDK_APP_ID | 短信应用 SDK AppId |
| TENCENT_SMS_TEMPLATE_ID | 短信模板 ID |
| TENCENT_SMS_SIGN_NAME | 短信签名 |

### 8.2 获取方式

1. **SecretId/SecretKey**: [腾讯云访问密钥](https://console.cloud.tencent.com/cam/capi)
2. **SDK AppId**: [短信控制台 → 应用管理](https://console.cloud.tencent.com/sms/smsList)
3. **模板ID/签名**: [短信控制台 → 模板管理](https://console.cloud.tencent.com/sms/smsList)
