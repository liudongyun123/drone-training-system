# 营销工具数据库集合创建指南

## 需要创建的数据库集合

营销工具功能需要以下额外的数据库集合：

### 1. userCoupons（用户优惠券）

**集合名称**: `userCoupons`

**用途**: 存储用户领取的优惠券信息

**字段结构**:
```json
{
  "_id": "string (自动生成)",
  "userId": "string (必需)",
  "couponId": "string (必需)",
  "couponCode": "string (必需)",
  "coupon": "object (优惠券详情)",
  "status": "string (unused/used/expired)",
  "orderId": "string (可选)",
  "obtainedAt": "string (ISO时间)",
  "usedAt": "string (可选)",
  "expiresAt": "string (必需)",
  "createdAt": "string (ISO时间)"
}
```

**安全规则**: `CUSTOM` - 用户只能操作自己的优惠券

```javascript
// 安全规则配置
{
  "read": "auth.openid == doc.userId",
  "write": "auth.openid == doc.userId"
}
```

**索引建议**:
- `userId` (普通索引)
- `couponId` (普通索引)
- `status` (普通索引)
- 组合索引: `[userId, couponId]`
- 组合索引: `[userId, status, expiresAt]`

## 创建集合步骤

### 方法1: 使用CloudBase控制台

1. 登录 [CloudBase控制台](https://tcb.cloud.tencent.com)
2. 选择环境: `rcwljy-5ghmq2ex26764978`
3. 进入"数据库" -> "文档数据库"
4. 点击"新建集合"
5. 输入集合名称: `userCoupons`
6. 点击"确定"

### 方法2: 使用CloudBase MCP工具

```javascript
// 调用MCP工具创建集合
await mcp_call_tool({
  serverName: "cloudbase",
  toolName: "writeNoSqlDatabaseStructure",
  arguments: {
    action: "createCollection",
    collectionName: "userCoupons"
  }
});
```

### 方法3: 使用云函数

创建云函数 `createCollection`:

```javascript
// cloudfunctions/createCollection/index.js
const cloud = require('@cloudbase/node-sdk');
const app = cloud.init({
  env: cloud.getCurrentEnv()
});
const db = app.database();

exports.main = async (event) => {
  const { collectionName } = event;
  
  try {
    await db.createCollection(collectionName);
    return {
      code: 0,
      message: `集合 ${collectionName} 创建成功`
    };
  } catch (error) {
    return {
      code: -1,
      message: error.message
    };
  }
};
```

调用云函数:

```javascript
await app.callFunction({
  name: 'createCollection',
  data: {
    collectionName: 'userCoupons'
  }
});
```

## 配置安全规则

创建集合后，需要配置安全规则：

### 使用MCP工具配置

```javascript
await mcp_call_tool({
  serverName: "cloudbase",
  toolName: "writeSecurityRule",
  arguments: {
    collectionName: "userCoupons",
    ruleType: "CUSTOM",
    ruleContent: {
      read: "auth.openid == doc.userId",
      write: "auth.openid == doc.userId"
    }
  }
});
```

### 使用CloudBase控制台配置

1. 进入集合详情页
2. 点击"权限设置"
3. 选择"自定义权限"
4. 配置读写规则:
   - 读权限: `auth.openid == doc.userId`
   - 写权限: `auth.openid == doc.userId`
5. 点击"保存"

## 注意事项

1. **集合名称区分大小写**: `userCoupons` 和 `usercoupons` 是不同的集合

2. **安全规则生效时间**: 安全规则配置后需要2-5分钟才能生效

3. **索引创建时间**: 创建索引可能需要几分钟时间，特别是数据量较大时

4. **性能优化**: 
   - 频繁查询的字段建议创建索引
   - 复合查询字段建议创建复合索引
   - 避免在大数据集上使用 `limit(0)` 获取所有数据

5. **数据迁移**: 
   - 如果已有数据，迁移前请备份
   - 建议使用云函数进行批量数据迁移

## 故障排查

### 问题1: 集合创建失败

**原因**: 集合名称不符合规范或已存在

**解决方案**:
- 检查集合名称是否只包含字母、数字、下划线
- 检查集合是否已存在
- 删除同名集合后重新创建

### 问题2: 安全规则不生效

**原因**: 规则配置错误或未生效

**解决方案**:
- 检查规则语法是否正确
- 等待2-5分钟后重试
- 使用测试功能验证规则

### 问题3: 查询性能差

**原因**: 缺少索引或索引未创建成功

**解决方案**:
- 检查索引是否创建成功
- 为查询字段创建索引
- 优化查询条件
