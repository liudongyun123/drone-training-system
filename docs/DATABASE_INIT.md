# 数据库初始化指南

## 快速开始

你的项目现在可以直接使用，按以下步骤初始化数据库：

---

## 第一步：部署云函数

在 CloudBase 控制台：

1. 打开 [CloudBase 控制台](https://cloudbase.net)
2. 选择你的环境
3. 进入「云函数」页面
4. 点击「新建云函数」
5. 函数名：`init-database`
6. 上传 `cloudfunctions/init-database` 目录
7. 点击「部署」

---

## 第二步：执行初始化

部署成功后：

1. 点击「测试」按钮
2. 输入 `{}` 作为测试参数
3. 点击「执行」

执行成功后返回：
```json
{
  "success": true,
  "message": "数据库初始化完成",
  "summary": {
    "courses": "4/4",
    "classes": "2/2",
    "products": "6/6",
    "categories": "3/3",
    "teachers": "4/4"
  }
}
```

---

## 第三步：验证数据

进入「数据库」页面，你会看到以下集合已创建：

| 集合名称 | 数据量 | 说明 |
|----------|--------|------|
| courses | 4条 | 课程数据 |
| classes | 2条 | 培训班数据 |
| products | 6条 | 商品数据 |
| product_categories | 3条 | 商品分类 |
| teachers | 3条 | 教师数据 |

---

## 第四步：刷新页面

**Web 端**：刷新浏览器，首页会显示课程、培训班、商品

**小程序**：
1. 微信开发者工具点击「编译」
2. 首页会显示真实数据

## 后续操作

初始化完成后，你可以：

1. 在后台管理修改/添加数据
2. 在数据库直接编辑数据
3. 再次执行云函数覆盖数据

---

## 常见问题

### Q: 执行云函数报错？
A: 检查环境是否正确初始化，确保已开通云函数服务。

### Q: 数据显示不出来？
A: 刷新页面，或检查数据库集合是否创建成功。

### Q: 想添加更多数据？
A: 修改 `cloudfunctions/init-database/index.js` 中的数据数组，重新部署执行。
