# 无人机培训管理系统 - 数据库设计文档

## 项目概述

无人机线上线下相结合的培训管理系统，分为客户端与后台管理端。

### 核心功能模块

**客户端**
- 页面展示（首页、课程列表、课程详情）
- 热门课程推荐
- 课程报名
- 调课功能

**后台管理端**
- 教师管理系统（上课安排表、出勤统计导出）
- 培训人员管理系统
- 课程管理系统
- 财务管理系统（订单管理、数据统计）
- 日常运营管理系统（宣传片管理）

---

## 技术架构

- **后端**: CloudBase 云函数 (Node.js)
- **数据库**: CloudBase NoSQL 文档数据库
- **前端**: React + Vite
- **SDK**: @cloudbase/js-sdk (客户端) + @cloudbase/node-sdk (云函数)

## 索引设计建议

### 常用查询索引

```javascript
// users 集合索引
db.users.createIndex({ phone: 1 }, { unique: true })
db.users.createIndex({ userType: 1, status: 1 })

// courses 集合索引
db.courses.createIndex({ status: 1, sort: -1 })
db.courses.createIndex({ category: 1, level: 1 })
db.courses.createIndex({ teacherId: 1 })
db.courses.createIndex({ isHot: -1, enrolledCount: -1 })

// enrollments 集合索引
db.enrollments.createIndex({ userId: 1, status: 1 })
db.enrollments.createIndex({ courseId: 1, status: 1 })
db.enrollments.createIndex({ enrollmentNo: 1 }, { unique: true })

// course_schedules 集合索引
db.course_schedules.createIndex({ courseId: 1, date: 1 })
db.course_schedules.createIndex({ teacherId: 1, date: 1 })
db.course_schedules.createIndex({ date: 1, startTime: 1 })

// attendance_records 集合索引
db.attendance_records.createIndex({ scheduleId: 1 })
db.attendance_records.createIndex({ userId: 1, date: -1 })

// orders 集合索引
db.orders.createIndex({ userId: 1, createTime: -1 })
db.orders.createIndex({ orderNo: 1 }, { unique: true })
db.orders.createIndex({ paymentStatus: 1, createTime: -1 })

// user_progress 集合索引
db.user_progress.createIndex({ userId: 1 })
db.user_progress.createIndex({ courseId: 1, progressPercentage: -1 })
```

## 注意事项

1. **安全优先**：所有写操作通过云函数，前端只能读取
2. **索引优化**：根据实际查询频率创建合适的索引
3. **数据分页**：所有列表查询都应支持分页
4. **字段扩展**：预留一些扩展字段以应对未来需求
