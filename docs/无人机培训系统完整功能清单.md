## 技术实现

### 技术栈
- **前端框架**：React 18 + TypeScript
- **UI 框架**：Tailwind CSS
- **路由管理**：React Router v6
- **后端服务**：CloudBase 云函数（Node.js）
- **数据库**：CloudBase NoSQL 文档数据库
- **SDK**：@cloudbase/js-sdk（客户端） + @cloudbase/node-sdk（云函数）

### 核心架构
- **服务层**：封装所有 CloudBase 调用，统一错误处理
- **组件层**：函数式组件 + Hooks，状态管理清晰
- **路由管理**：React Router，页面路由清晰
- **数据流**：单向数据流，通过 props 传递数据

### 数据库设计
- **11 个核心集合**
  - users（用户）
  - user_profiles（学员档案）
  - teacher_profiles（教师档案）
  - courses（课程）
  - course_schedules（排课）
  - enrollments（报名记录）
  - schedule_changes（调课申请）
  - attendance_records（出勤记录）
  - promotions（宣传素材）
  - statistics_daily（每日统计）
  - statistics_teacher（教师统计）

### 权限控制
- **数据库安全规则**
  - 用户数据隔离（只能访问自己的数据）
  - 管理员权限控制
  - 前端只能读取，写操作通过云函数
  - 公开数据（课程、教师）可读取
