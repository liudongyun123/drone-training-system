## 技术实现

### 技术栈
- **前端框架**：React + TypeScript
- **UI框架**：Tailwind CSS
- **后端服务**：CloudBase 云函数
- **数据库**：CloudBase NoSQL 文档数据库
- **SDK**：@cloudbase/js-sdk（客户端） + @cloudbase/node-sdk（云函数）

### 架构设计
- **服务层**：封装所有 CloudBase 调用，统一错误处理
- **组件层**：函数式组件 + Hooks，状态管理清晰
- **路由管理**：React Router，页面路由清晰
- **数据流**：单向数据流，通过 props 传递数据
