## 实施顺序

### 第一步：扩展现有集合
1. 扩展 users 集合（添加 userType, realName 等字段）
2. 扩展 courses 集合（添加 type, level, location 等字段）

### 第二步：创建新集合
1. 创建 user_profiles, teacher_profiles
2. 创建 course_schedules
3. 创建 enrollments, schedule_changes, attendance_records
4. 创建 statistics_daily, statistics_teacher

### 第三步：配置安全规则
1. 为所有集合配置安全规则
2. 等待 2-5 分钟规则生效

### 第四步：创建索引
1. 为所有集合创建必要的索引
2. 验证索引创建成功
