## 需要手动创建的集合

### 方法一：通过 CloudBase 控制台创建（推荐）

访问数据库控制台：
https://tcb.cloud.tencent.com/dev?envId=rcwljy-5ghmq2ex26764978#/db/doc

创建以下集合：

1. **user_profiles** - 学员档案表
2. **teacher_profiles** - 教师档案表
3. **course_schedules** - 课程排课表
4. **enrollments** - 报名记录表
5. **schedule_changes** - 调课申请表
6. **attendance_records** - 出勤记录表
7. **promotions** - 宣传素材表
8. **statistics_daily** - 每日统计表
9. **statistics_teacher** - 教师统计表

### 创建步骤

1. 点击"新建集合"
2. 输入集合名称（如 `user_profiles`）
3. 选择权限：暂选"自定义规则"
4. 点击"确定"

## 配置安全规则

创建集合后，需要为每个集合配置安全规则。

### users 集合

```json
{
  "read": "auth != null && (doc.userType == 'admin' || doc._openid == auth.openid)",
  "write": "auth != null && (doc.userType == 'admin' || doc._openid == auth.openid)"
}
```

### user_profiles 集合

```json
{
  "read": "auth != null && (doc.userId == auth.openid || auth.userType == 'admin')",
  "write": "false"
}
```

### teacher_profiles 集合

```json
{
  "read": "auth != null",
  "write": "false"
}
```

### courses, course_schedules 集合

```json
{
  "read": "auth != null",
  "write": "false"
}
```

### enrollments 集合

```json
{
  "read": "auth != null && (doc.userId == auth.openid || auth.userType == 'admin')",
  "write": "false"
}
```

### schedule_changes 集合

```json
{
  "read": "auth != null && (doc.userId == auth.openid || doc.approverId == auth.openid || auth.userType == 'admin')",
  "write": "false"
}
```

### attendance_records 集合

```json
{
  "read": "auth != null && (doc.userId == auth.openid || doc.teacherId == auth.openid || auth.userType == 'admin')",
  "write": "false"
}
```

### promotions, statistics_daily, statistics_teacher 集合

```json
{
  "read": "auth != null && auth.userType == 'admin'",
  "write": "false"
}
```

### 配置步骤

1. 在数据库控制台，点击集合名称
2. 点击"安全规则"标签
3. 点击"编辑规则"
4. 粘贴对应的安全规则代码
5. 点击"保存"
6. 等待 2-5 分钟让规则生效

## 验证集合创建

创建完成后，可以在云函数中调用 admin 服务来验证：

console.log(result)

## 注意事项

1. 集合名称必须完全匹配（区分大小写）
2. 安全规则配置后需要等待 2-5 分钟生效
3. 所有写操作通过云函数 `admin` 执行
4. 前端只能读取数据，不能直接写入
5. 首次创建集合后，建议创建索引以优化查询性能
